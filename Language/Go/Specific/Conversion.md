
# String to Byte array

```go
bytes := []byte(str)
```

Byte array to String

```go
str := string(bytes)
```

# Easy String Format Reference

```
%s -> string
%q -> string wrapped with quotes
%d -> base 10 integer
%T -> type of the object
%t -> boolean
%f -> floating point
%p -> pointer
%v -> value in its default format
%+v -> include field names
%#v -> include field names and type
```
